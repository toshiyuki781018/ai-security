## 責任主体形成の原則（暫定）

| Rule | 内容 |
|------|------|
| Rule-01 | 責任主体が存在しない業務は、運用できない |
| Rule-02 | 観測点のないリスクは、管理できない |
| Rule-03 | 制御点のない問題は、改善できない |
| Rule-04 | 責任分界点のない組織では、問題は宙に浮く |
| Rule-05 | AIは、責任主体の代替にならない |
